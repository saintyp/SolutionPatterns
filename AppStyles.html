App Styles

In this section we have concentrated on three main styles.

Component based/microservices

style that decomposes your applications into the separate components that it needs to function.

Message-Bus/Event driven

A style were each part of the application publishes events to centralised bus that other parts of the application can consume.

Data-analytics

A style that requires you to mine existing data for either MI or customer analytics.

What does good look like for these application styles

Whilst there are options of how to implement these styles these reference architectures will give you a good idea of best practice implementations.
