What does good look like for security?

The ultimate aim is to secure the solution against malicious threats. This is ultimately a defensive in depth design and you need to consider baking in this throughout your solution. This will include • border security • secure coding • Encrypting data at rest and in transit • Secure access to the application or infrastructure • Using least privilege access process
