How to build cloud solutions

A solution for a cloud application does not solely consist of how the application is architected. It is important to define how the system interacts with other elements.

The product requirements will lead to you making choices around the style of application you are trying to produce and how you need to store/retrieve data. These factors flow into an application design which then needs to interact with the cloud eco system around it so that we have a full solution.
