Integration Patterns

Integration patterns are common requirements that all Cloud application solutions will require. They range from specifying requirements for data encryption, key management, certificate management to logging and integration with on premise toolsets.

The integration patterns may define the overall architecture that you will be using. However as a solution architect how to implement the requirements of the pattern may be more of use. In these cases each integration pattern is accompanied by an implementation guide with corresponding code for implementing in AWS.
